# Gas Relay Signer

This is a relay signer that can generate and sign meta-transactions.

## Prerequisites

* Go 1.12+ installation or later
* **GOPATH** environment variable is set correctly

## Install

```
$ git clone https://github.com/lacchain/gas-relay-signer

$ cd gas-relay-signer
$ go build
```

## Run

```
$ gas-relay-signer
